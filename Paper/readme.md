# Paper — IDRL-LOB

This folder contains the research manuscript and supporting material for:

**_Estimating Strategic Delays in Limit Order Books via EM–MaxEnt IRL_**  
**Working Draft (v0.9) — Preprint forthcoming**

## Files
- **main.pdf** — Current draft of the paper  
- **main.tex** — LaTeX source  
- **references.bib** — Bibliography  
- **figures/** — All diagrams used in the manuscript  
- **supplementary/** — Full derivations and extended mathematical details

## About
This paper develops a complete framework for modelling Limit Order Books (LOBs) as asynchronous decision processes and recovering trader delays and strategies using **EM–MaxEnt Inverse Reinforcement Learning**.

A formal preprint will be released after incorporating feedback from faculty reviewers.
